瘦吧App - API调用

本文档提供瘦吧App的页面调用API接口

说明

瘦吧App提供两个服务，一个主API服务，一个用户服务

Master Base URL: http://api.shouba.cn:80

User Base URL: http://api.shouba.cn:81
